# Senparc.AI Console Sample Usage

Open the current solution and build or publish the console project.

`Run directly` starts the program and displays the console menu.

`Publish` outputs the program to a specified directory. You can then run it from the command line:

1. Open the output folder.

2. Edit `appsettings.json` with an editor or Notepad, then fill in the required AI model settings.

If you do not currently have a model available, you can add the WeChat account `SenparcWeChat` to get free quota for testing.

3. Start the program directly:

```powershell
Senparc.AI.Samples.Consoles.exe
```

You can also open the publish directory from the command line and run:

```powershell
dotnet Senparc.AI.Samples.Consoles.dll
```
